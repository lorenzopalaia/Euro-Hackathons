# <p align="center">🇪🇺🚀 EURO HACKATHONS</p>

### <p align="center">A Modern Web Application for European Hackathons</p>

## 🚀 Tech Stack

- **Frontend**: Next.js 15 with React and TypeScript
- **Styling**: Tailwind CSS
- **Data Fetching**: Next.js API Routes
- **Database**: Supabase (for storing hackathon data)

## 🌟 Features

- **Real-time Hackathon Updates**: Live data synchronization
- **Advanced Filtering**: Search by date, location, and topics
- **Responsive Design**: Mobile-first approach
- **Dark/Light Mode**: Theme customization
- **Contribution System**: Community-driven content updates

## 🛠️ Installation

```bash
# Clone the repository
git clone https://github.com/lorenzopalaia/euro-hackathons.git

# Install dependencies
npm install

# Run development server
npm run dev
```
